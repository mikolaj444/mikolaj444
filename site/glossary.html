Glossary

Attribute
used to identify an element, contains a name and a value
HTML Head
Non-visible aspects of the website, used to identify language, favicon, title, author etc.
Block Level Elements
Create a physical block, appear in their own space/line. Can contain other elements within.
Inline Elements
Show up INLINE with the content, and wrap at the end. Cannot contain elements within.
Absolute URLS
The entire URL including (https:) as well as domain name
Relative URLS
A path relative to current webpage, links only within the website (for example this site linking to this sub page)
